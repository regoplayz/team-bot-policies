# Privacy Policy for Team Bot

This Discord bot (“the Bot”) does not collect, store, or share any personal user data.

## Data the Bot Uses
The Bot only uses non-personal server configuration data:
- Server ID
- Channel ID
- Role ID
- Team names and team configuration

This information is stored solely to enable the Bot’s features (creating teams, roles, and channels).

## No Personal Data Collection
The Bot does NOT:
- Collect message content
- Track users
- Log personal information
- Share any data with third parties

## Data Removal
If the Bot is removed from a server, all related configuration data becomes inactive.

If you need manual data deletion, contact the bot owner.
